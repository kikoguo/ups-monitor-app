# UPS WiFi智能监控系统 - 快速开始指南

## 项目概述

这是一个完整的UPS智能监控解决方案，包含：
- **ESP8266固件**：连接UPS与WiFi网络
- **Flutter APP**：移动端监控应用（Android/iOS）

## 硬件准备

### 需要的硬件
1. ESP8266开发板（NodeMCU或Wemos D1 Mini）
2. UPS设备（带串口通信接口）
3. USB数据线（用于烧录固件）
4. 杜邦线（连接ESP8266与UPS）

### 接线图

```
ESP8266 (NodeMCU)     UPS主板
----------------     --------
D5 (GPIO14)   ────►   TX  (数据发送)
D6 (GPIO12)   ────►   RX  (数据接收)
GND           ────►   GND
```

**注意**：某些UPS使用5V TTL电平，需要电平转换模块！

## 第一步：烧录ESP8266固件

### 1.1 安装Arduino IDE
1. 下载Arduino IDE：https://www.arduino.cc/en/software
2. 安装并打开Arduino IDE

### 1.2 安装ESP8266开发板
1. 打开 Arduino IDE
2. 点击 `文件` → `首选项`
3. 在"附加开发板管理器网址"中添加：
   ```
   http://arduino.esp8266.com/stable/package_esp8266com_index.json
   ```
4. 点击 `好`
5. 点击 `工具` → `开发板` → `开发板管理器`
6. 搜索"ESP8266"并安装

### 1.3 安装库
1. 点击 `项目` → `加载库` → `管理库`
2. 安装以下库：
   - **ArduinoJson** by Benoit Blanchon

### 1.4 烧录固件
1. 打开文件 `esp8266/ups_wifi_server/ups_wifi_server.ino`
2. 修改WiFi配置（约第15-16行）：
   ```cpp
   const char* ssid = "你的WiFi名称";
   const char* password = "你的WiFi密码";
   ```
3. 选择开发板：`工具` → `开发板` → `NodeMCU 1.0 (ESP-12E Module)`
4. 选择端口：`工具` → `端口` → `COM3`（根据实际选择）
5. 点击上传按钮 ▶

### 1.5 验证固件
1. 打开串口监视器（波特率115200）
2. 应该看到类似输出：
   ```
   正在连接WiFi: 你的WiFi名称
   WiFi已连接!
   IP地址: 192.168.1.100
   HTTP服务器已启动
   ```

## 第二步：安装Flutter APP

### 2.1 安装Flutter SDK

#### Windows:
1. 下载Flutter SDK：https://flutter.dev/docs/get-started/install/windows
2. 解压到 `C:\flutter`（不要放在需要管理员权限的目录）
3. 将 `C:\flutter\bin` 添加到系统PATH
4. 打开命令提示符，运行 `flutter doctor`

#### macOS:
1. 下载Flutter SDK
2. 解压到 `/Users/你的用户名/flutter`
3. 添加到PATH：`export PATH="$PATH:/Users/你的用户名/flutter/bin"`
4. 运行 `flutter doctor`

#### Linux:
1. 下载Flutter SDK
2. 解压到 `/opt/flutter`
3. 添加到PATH
4. 运行 `flutter doctor`

### 2.2 运行APP

```bash
# 进入APP目录
cd ups-project/flutter_app/ups_monitor_app

# 获取依赖
flutter pub get

# 运行APP（调试模式）
flutter run
```

### 2.3 构建发布版本

#### Android APK:
```bash
flutter build apk --release
# APK文件在: build/app/outputs/flutter-apk/app-release.apk
```

#### iOS:
```bash
flutter build ios --release
# 需要Mac和Xcode
```

## 第三步：使用APP

### 3.1 首次使用
1. 打开APP
2. 点击右下角的 🔍 扫描按钮
3. APP会搜索局域网内的UPS设备
4. 发现设备后，点击"添加"

### 3.2 手动添加设备
如果扫描不到：
1. 点击右下角的 ➕ 按钮
2. 输入设备IP（查看ESP8266串口输出的IP地址）
3. 输入设备名称
4. 点击"添加"

### 3.3 查看监控数据
1. 点击设备卡片进入监控页面
2. 实时查看：
   - 输出电压、电流、功率
   - 电池电量
   - 负载百分比
   - 剩余运行时间

### 3.4 控制UPS
1. 点击右上角的 ⚙️ 按钮进入控制面板
2. 可执行操作：
   - 关机
   - 重启
   - 自检
   - 旁路模式
   - 静音/取消静音

## 常见问题

### Q1: ESP8266无法连接WiFi
**解决**：
- 确认SSID和密码正确
- 确保WiFi是2.4GHz（ESP8266不支持5GHz）
- 检查WiFi是否开启了AP隔离

### Q2: APP无法发现设备
**解决**：
- 确保手机和ESP8266在同一WiFi网络
- 检查防火墙设置
- 手动输入IP地址添加设备

### Q3: 无法获取UPS数据
**解决**：
- 检查TX/RX接线是否正确
- 确认波特率匹配（默认9600）
- 检查是否需要电平转换

### Q4: APP编译失败
**解决**：
```bash
flutter clean
flutter pub get
flutter pub upgrade
```

## API接口参考

### 获取UPS状态
```
GET http://<IP>/status
```

### 发送控制命令
```
POST http://<IP>/control
Content-Type: application/json

{"command": "shutdown"}  // shutdown | restart | test | bypass
```

### 获取设备信息
```
GET http://<IP>/info
```

### 健康检查
```
GET http://<IP>/health
```
