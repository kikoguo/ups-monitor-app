### 🎯 已实现的功能

#### ESP8266固件 ✅
- WiFi连接与自动重连
- HTTP服务器（RESTful API）
- UPS数据读取（支持Q1命令）
- 控制命令发送
- LED状态指示

#### Flutter APP ✅
- 📱 **首页**：设备列表展示、自动刷新
- 🔍 **设备发现**：局域网自动扫描、手动添加
- 📊 **监控页面**：实时数据、仪表盘图表、状态指示
- ⚡ **控制面板**：关机、重启、自检、旁路、静音
- 💾 **本地存储**：设备列表持久化
- 🎨 **主题支持**：浅色/深色模式自动切换

### 🚀 如何开始

#### 1. 烧录ESP8266固件
打开 `ups-project/esp8266/ups_wifi_server/ups_wifi_server.ino`，修改WiFi配置后上传

#### 2. 运行Flutter APP
```bash
cd ups-project/flutter_app/ups_monitor_app
flutter pub get
flutter run
```
